GMS_Core Install Instructions//

//Step 1;

	Download GMSCore - Master and unextract the file to your documents.
	
//Step 2;

	Open the file "CfgGMSCore.hpp" and adjust the "hunt player" options to your prefrence (Remember to save any changes!).
	
//Step 3;

	Rename the "\x\addons\GMSCore-master" file to "\x\addons\GMSCore" and PBO it.
	
//Step 4;

	Add the "\x\addons\GMSCore.pbo" file to your server addons (@Exileserver\addons or @Epochhive\addons)

	Restart your server and proceed to install anyother GRG scripts which depend on GMSCore 
	
	If you run into any issues check your client/server RPT for errors.
